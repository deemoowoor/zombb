Zombb Rails message board


Quick Start
===========

0. You need Ruby 1.9+ at minimum

1. Install dependencies with

    $ bundle install

2. Migrate DB

    $ rake db:migrate

3. Run server with

    $ rails s

4. Accessible by default at ``http://localhost:3000/``


Running unittests
=================

Unittests are run with

    $ rails test
